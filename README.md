# 📚 Java Library Management System

A console-based Library Management System built in Java that demonstrates Object-Oriented Programming, HashMap data storage, CRUD operations, file handling, and exception handling.

## ✨ Features

* ➕ Add new books
* 📖 Display all books
* 🔍 Search books by Book ID
* 📕 Borrow books
* 📗 Return books
* 💾 Automatically save library data to a file
* 🔄 Automatically load saved data when the application starts
* 🛡️ Input validation and exception handling
* 🚫 Prevent duplicate Book IDs

## 🛠️ Technologies Used

* Java
* Java Collections Framework (`HashMap`)
* Java File I/O (`FileWriter`, `FileReader`, `BufferedReader`)
* Exception Handling

## 🚀 How to Run

1. Clone this repository.
2. Open the project in your preferred Java IDE (VS Code, IntelliJ IDEA, Eclipse, etc.).
3. Compile and run `Main.java`.
4. Use the menu to manage the library.
